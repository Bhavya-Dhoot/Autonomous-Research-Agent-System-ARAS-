# Research Planner Skill

## Core Task

Given a research topic, produce a **STRICT JSON** plan with:

- **hypothesis** ‚Äî 1 sentence, falsifiable claim
- **questions** ‚Äî 3‚Äì5 specific research questions
- **experiments** ‚Äî 3 experiment names matching the domain
- **metrics** ‚Äî list of measurable outcomes
- **outline** ‚Äî list of paper section titles
- **keywords** ‚Äî 8‚Äì12 search terms for scraping
- **domain** ‚Äî one of: `nlp`, `computer_vision`, `reinforcement_learning`, `general_ml`, `theory`, `systems`, `bioinformatics`, `other`

## Output Format

Return **JSON only**. No markdown. No preamble. No explanation.

```json
{
  "hypothesis": "We hypothesize that ...",
  "questions": ["Q1", "Q2", "Q3"],
  "experiments": ["exp1", "exp2", "exp3"],
  "metrics": ["accuracy", "f1", "runtime"],
  "outline": ["Abstract", "Introduction", "Related Work", "Methodology", "Experiments", "Results", "Discussion", "Conclusion"],
  "keywords": ["keyword1", "keyword2", "...", "keyword10"],
  "domain": "general_ml"
}
```

## Quality Criteria

- Hypothesis **must be falsifiable** ‚Äî contains "we hypothesize", "we propose", or "we demonstrate"
- Keywords must include both **general terms** AND **specific technical terms** (e.g., both "machine learning" and "gradient boosting calibration")
- Domain must match the most prominent keywords
- Experiments must be concrete and runnable in Python under 5 minutes
- Output must be valid parseable JSON ‚Äî no trailing commas, no comments

## Example

**Input**: "Calibration of Confidence Scores in Text Classification Models"

**Output**:
```json
{
  "hypothesis": "We hypothesize that temperature scaling with Platt calibration outperforms raw softmax probabilities for confidence calibration in transformer-based text classifiers.",
  "questions": ["How does temperature scaling compare to isotonic regression for confidence calibration?", "What is the impact of dataset size on calibration quality?", "Do multilingual models exhibit worse calibration than monolingual models?"],
  "experiments": ["baseline_calibration", "temperature_scaling_sweep", "cross_domain_calibration"],
  "metrics": ["expected_calibration_error", "brier_score", "reliability_diagram_auc", "accuracy"],
  "outline": ["Abstract", "Introduction", "Related Work", "Methodology", "Experiments", "Results", "Discussion", "Conclusion"],
  "keywords": ["confidence calibration", "text classification", "temperature scaling", "Platt scaling", "expected calibration error", "transformer", "softmax probability", "isotonic regression", "reliability diagram", "Brier score"],
  "domain": "nlp"
}
```

## Eval Assertions

- **json_parseable**: output is valid JSON
- **has_all_keys**: all 7 required keys present (hypothesis, questions, experiments, metrics, outline, keywords, domain)
- **hypothesis_falsifiable**: contains "hypothesize" or "propose" or "demonstrate"
- **keyword_count**: 8 <= len(keywords) <= 12
- **domain_valid**: domain in ["nlp", "computer_vision", "reinforcement_learning", "general_ml", "theory", "systems", "bioinformatics", "other"]
- **questions_count**: 3 <= len(questions) <= 5

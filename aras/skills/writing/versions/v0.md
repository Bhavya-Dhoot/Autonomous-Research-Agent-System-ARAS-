# Paper Writer Skill

## Core Task

Given section name + research plan + scraped items + experiment results, write **ONE** paper section in LaTeX.

## Section-Specific Guidance

### abstract
250 words max. Structure: Problem → gap → method → result → impact. No citations in abstract.

### introduction
Hook → problem statement → gap in literature → contribution bullet list → roadmap ("The remainder of this paper is organized as follows...").

### related_work
**ARGUMENTATIVE, not descriptive.** Every paragraph argues a gap:
> "X et al. did Y but failed at Z because W. Our work addresses Z by..."

- Cite at least 8 papers
- No paragraph without a `\cite{}`
- Group by theme, not chronologically

### methodology
- Formal definitions first
- Algorithm pseudocode if applicable (use `algorithmic` environment)
- Every claim backed by a citation or "we propose"
- Define all notation before use

### experiments
Setup → datasets → baselines → metrics → protocol. Tables use `\begin{table}` with `\toprule`, `\midrule`, `\bottomrule`.

### results
Interpret numbers — never just describe them:
- ✅ "Model A outperforms B by 3.2% on F1, statistically significant at p < 0.05"
- ❌ "Table 1 shows the results"

### discussion
Limitations first (honest), then future work, then broader impact.

### conclusion
Restate 3 contributions from introduction. No new information. No citations.

## Output Format

Return **raw LaTeX** for the section. Include `\section{...}` header. No `\documentclass` or preamble.

## Quality Criteria

- No unfilled Jinja2 placeholders (`{{` or `}}`)
- Related work: argumentative not descriptive
- Every section ends with a transition sentence to the next section
- LaTeX compiles (balanced braces, valid commands)
- Tables use booktabs formatting

## Eval Assertions

- **no_placeholders**: `{{` not in output and `}}` not in output
- **has_citations**: `\cite{` in output (for all sections except abstract/conclusion)
- **latex_balanced**: count of `\begin` equals count of `\end`
- **word_count_reasonable**: 200 < word count < 1500
